## The Problem

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

Find the sum of all the multiples of 3 or 5 below the provided parameter value `number`

```js
function multiplesOf3and5(number) {
  return true;
}
multiplesOf3and5(1000);
```

Untuk problem pertama kita, kita diberi sebuah angka , kita diharuskan mencari angka dibawah angka tersebut yang merupakan kelipatan 3 atau 5 kemudian kita harus menjumlahkan angka angka tersebut.

---
## The Solution

```js
function multiplesOf3and5(number) {
  const arrNumber = []
  let i;

  for(i = 1; i < number; i++) {
    if(i % 3 === 0 || i % 5 === 0) {
      arrNumber.push(i)
    }
  }

  return arrNumber.reduce((accumulator, currentValue) => {
    return accumulator + currentValue
  });
} 

multiplesOf3and5(1000);
```

ini solusi yang levi buat, didalam function `multiplesOf3and5` levi membuat sebuah variable `arrNumber` dimana variable ini dibuat untuk menampung angka angka yang dicari.

kemudian levi membuat `for loop` dimana looping ini akan terus berjalan jika variable i masih lebih kecil dari nilai `number` kemudian levi membuat sebuah `conditional statement` didalam `for loop` tersebut dimana jika variable i % 3 atau i % 5 sama dengan 0 maka nilai i tersebut akan dimasukkan kedalam variable `arrNumber`

ketika for loop sudah selesai langkah terakhir adalah menghitung / menjumlahkan seluruh angka angka di variable `arrNumber`, levi menggunakan [Array.prototype.reduce()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/reduce) dan langsung mengembalikan nilai dari kalkulasi tersebut
